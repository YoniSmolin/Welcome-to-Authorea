Abstract

As RGB-D sensors like Kinect become cheaper and more popular, it is interesting to look into the benefits of constructing arrays of such sensors. This report deals with the technical aspect of assembling a sensor network, each node of which is comprised of a an RGB-D sensor powered with a local processing unit. All the nodes are synchronized to a central processing unit. Thus, the network can perform both distributed and centralized logic and data processing. Such an network can be used as an efficient platform for various kinds of computer vision applications, including room mapping, event detection and tracking.
